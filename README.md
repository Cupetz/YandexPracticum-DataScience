# [Проект по предсказанию эффективности золотодобычи](https://nbviewer.org/github/Cupetz/YanexPractucum-DataScience/blob/main/%D0%B8%D1%81%D1%85%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5%20%D1%84%D0%B0%D0%B9%D0%BB%D1%8B%20%D0%BF%D1%80%D0%BE%D0%B5%D0%BA%D1%82%D0%BE%D0%B2/1.%20%D0%9E%D0%B1%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%B5%20%D0%BC%D0%BE%D0%B4%D0%B5%D0%BB%D0%B8%20-%20%D0%BF%D1%80%D0%B5%D0%B4%D1%81%D0%BA%D0%B0%D0%B7%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%8D%D1%84%D1%84%D0%B5%D0%BA%D1%82%D0%B8%D0%B2%D0%BD%D0%BE%D1%81%D1%82%D0%B8%20%D0%B7%D0%BE%D0%BB%D0%BE%D1%82%D0%BE%D0%B4%D0%BE%D0%B1%D1%8B%D1%87%D0%B8.ipynb)
- в данном проекте обработана большая база сырых данных о золотодобывающих установках с пропусками, аномальными значениями, большим набором признаков (порядка 80)
- вычленены нужные обучающие и целевые признаки для каждой из двух моделей, проведена предобработка данных и обучены обе требуемые модели
- для обучения моделей о оценки их эффективности была задействована кросс-валидация, применено создание собственной метрики качества через make_score, для подбора оптмальных гиперпараметров задействован метод GridSearch. Для сравнения и поиска наиболее эффективной модели были обучены сразу две - случайного леса и логистической регрессии
